iOS static const extern 用法技巧

发表于 2016-01-28   |  

通俗的讲：

extern字段使用的时候，声明的变量为全局变量，都可以调用，也有这样一种比较狭义的说法：extern可以扩展一个类中的变量到另一个类中；

static声明的变量是静态变量，变量值改变过之后，保存这次改变，每次使用的时候都要读取一遍值；

const声明过得变量值是不可改变的，是readonly的属性，不可以改变变量的值。
